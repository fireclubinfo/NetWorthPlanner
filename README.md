# Net Worth Planner — releases

Download the latest Windows build from the [Releases page](https://github.com/fireclubinfo/NetWorthPlanner/releases) — grab `NetWorthPlanner-<version>-win-x64.zip`, extract it anywhere, and run `NetWorthPlanner.App.exe` (64-bit Windows 10 1809+, nothing to install).

The app, its terms, and purchase live at [fireclub.info](https://fireclub.info). Releases are code-signed (publisher: Daniel Yates).
