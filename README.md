# FIRE Club App — releases

Download the latest Windows build from the [Releases page](https://github.com/fireclubinfo/NetWorthPlanner/releases) — grab `FireClubApp-<version>-win-x64.zip`, extract it anywhere, and run `FireClubApp.exe` (64-bit Windows 10 1809+, nothing to install).

This repository hosts release downloads only — FIRE Club App is commercial, closed-source software, so the automatic "Source code" archives on each release contain just this README. The app, its terms, and purchase live at [fireclub.info](https://fireclub.info). Releases are code-signed (publisher: Daniel Yates).
